Finding the appropriate values

Report ID
This is found in the main url, right after "reports":
https://classic.warcraftlogs.com/reports/6xGYKBtgXzTvHacN
Fight ID
In the url, look for "fight=20"
Report Section
Which section of the report should be shown
Character
This is the Classic Logs character reference. I've populated several Crimson shamans, but you can input any character reference.
To find a reference, open a report, click [+] next to a character, then look in the url for "%2421697240.0.0.Shaman%24". You want the number directly after %24. These do not change from report to report, so once you've located it, make a note of it
Windfury Targets
Comma-separated list of names to show. This would be best populated with every meleer in the shaman's group. If you leave this empty, all characters that procced windfury will be shown.
Click on a Rogue or Warrior's name to add them to the list
